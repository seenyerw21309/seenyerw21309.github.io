### 浏览器首页被劫持成 hao123？Chrome/Edge 彻底清除教程（含流氓软件）

最近很多人遇到：
浏览器主页莫名变成 hao123，改完重启又回去；快捷方式被加网址,这篇从基础到顽固，一次性彻底解决。

### 一、最常见：修复桌面快捷方式（90% 问题在这）

右键 Chrome / Edge 图标 → 属性 → 快捷方式
看 目标(T) 一栏：

- 正常：

> "C:\Program Files\Google\Chrome\Application\chrome.exe"

- 被劫持：后面多了网址

> "......chrome.exe" http://www.hao123.com

**修复：**

>  1. 删除引号以外所有内容
>  2. 确定保存
> 3. 任务栏、开始菜单图标重新固定一遍

### 二、Chrome 浏览器恢复教程

**1.重置启动页**
⋮ → 设置 → 启动时
选择：打开新标签页，删除陌生网址

**2. 清理恶意扩展**
地址栏输入：
`chrome://extensions/`

卸载：主页助手、快速导航、不明插件

**3. 重置浏览器（终极）**
设置 → 高级 → 重置设置 → 还原为默认设置

### 三、Edge 浏览器恢复教程

**1. 重置启动页**
⋮ → 设置 → 启动、主页和新建选项卡
设为：新标签页

**2. 清理扩展**
地址栏输入：
 `edge://extensions/`
	
删除可疑扩展

**3. 重置 Edge**
设置 → 重置设置 → 还原为默认值

### 四、专项：被 scpt / 脚本 / 封装工具劫持（顽固必看）

这类劫持会自动改回主页，必须清理脚本和计划任务。

**1. 结束恶意进程**

任务管理器 → 详细信息
**结束：**

- wscript.exe
- cscript.exe
- 可疑 cmd.exe

**2. 删除开机启动脚本**

Win + R 输入：
`shell:startup`
**删除：**

- *.bat
- *.vbs
- 带“修复”“主页”“导航”的脚本

### 五、新增：流氓软件彻底卸载教程

**（一）正常卸载**

> 1. 打开 设置 → 应用 → 已安装的应用
> 2. 按 安装日期 排序
> 3. 卸载这些软件

**（二）删不掉？用强制卸载**

> 1. 打开 控制面板 → 程序和功能
> 2. 右键流氓软件 → 卸载/更改
> 3. 如果弹窗阻止、闪退，继续下面步骤

**（三）停止服务再删**

> 1. Win + R → services.msc
> 2. 找到对应软件的服务
> 3. 停止 → 禁用
> 4. 再回去卸载

**（四）强力清理工具（推荐）**

- Geek Uninstaller
- IObit Uninstaller
- AdwCleaner（专门清浏览器劫持）ivers\etc\hosts

### 【结尾建议】
• 安装软件一律 自定义安装
• 取消所有捆绑、锁定主页、修改导航
• 不使用不明破解、封装、脚本工具
